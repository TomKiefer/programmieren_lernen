Start

Was ist C# ?

C# ist eine objektorientierte Programmiersprache aus der .NET Plattform von Microsoft. Sie ist eine der Hauptsprachen für Windows- , Web- , Cloud- und Spiele-Anwendungen. Sie ist im Jahr 2000 erschienen und läuft auf dem .NET Framework oder .NET (Core). Ihre Syntax ist ähnlich wie bei C, C++ oder Java.

Anwendungsgebiete

  • Desktop Apps
  • Web-Anwendungen
  • Spieleentwicklung (mit Unity)
  • Cloud-Services
  • Mobile Apps

Wichtige Merkmale

  • Objektorientiert : nutzt z.B. Klassen, Vererbung und Polimorphie
  • Typensicher : erkennt viele Fehler bereits während des kompilierens
  • Garbage Collection : automatisches Sicherheitsmanegement
  • Async / Await : einfache nebenläufige Programmierung
  • Cross-Plattform : läuft mit .NET auf Windows, Linux und macOS

Grundlagen

Erstes Programm


using System;

class Program
{
    static void main()
    {
        Console.WriteLine("Hallo Welt!")
    }
}

Dieses Programm ist das Hallo Welt Programm, welches in fast allen Sprachen das erstes-Programm ist, was man codet. Dieses Hallo Welt Programm gibt in der Console folgendes aus :

Hallo Welt!

Erklärung

  • using System → Importiert den System-Namespace
  • class Program → Definiert eine Klasse mit dem Namen Program
  • static void main() → Einstiegspunkt des Programms / Beginn der Ausführung
  • Console.WriteLine("Hallo Welt!") → Gibt den Text : Hallo Welt! in der Konsole aus

Variablen


int alter = 25;
string name = "Anna";
double gehalt = 3500.50;
bool istVolljährig = true;

  • int = Ganze Zahl
  • string = Text / Zeichenkette
  • double = Gleitkommazahl
  • bool = Wahrheitswert (true / false)
  • float = einfache Kommazahl
  • decimal = sehr genaue Kommazahl
  • char = einzelnes Zeichen
  • byte = 8-Bit-Zahl

Merke:
int, string, double, ... sind Datentypen!
Sie bestimmen welche Art von Wert eine Variable annehmen kann! 


Bildung einer Variable:
datentyp name = wert;

Variablen sind also Speicherplätze für Werte verschiedener Art. Diese Art der Werte wird durch einen Datentypen vor dem Variablen-Namen bestimmt. Nach dem Namen kommt der Zuweisungsoperator "=" und dann der Wert der Variable. Eine Variable funktioniert nur, wenn der Wert zum vorher angegebenen Datentyp passt. So würde " int name = "Anna" " nicht funktionieren, da dass Programm eine Ganze Zahl erwartet und einen Text bekommt.

Konstanten verwenden

Eine Konstante ist ähnlich wie eine Variable ein gespeicherter Wert. Jedoch sind Konstanten schon im Code festgelegt und können während der Ausführung des Programms oder später im Code keinen anderen Wert zugewiesen bekommen. 


const double Pi = 3.14159265359;
const int MaxAlter = 120;
const string Begrüßung = "Wilkommen!";

Merke:
Bei Variablen, Constanten und generell überall im Code gilt, dass ein Komma immer ein Punkt "." ist. Speichert man Nachkomma-Zahlen mit einem Komma "," könnte das Probleme und Fehlermeldungen oder ungewollt Teile im Code ausführen.
Bsp.:
if pi < 3.5 {} → führt den Code nur aus, wenn Pi kleiner als 3,5 ist
if pi < 3,5 {} → führt den Code nur aus, wenn Pi kleiner als 35 ist 


Bildung einer Konstanten: 
const datentyp name = wert;

Kostanten akzeptieren außerdem nur einfache Datentypen wie int, double, char, string, bool, enum, usw.

Konstanten in Klassen


Deklarieren einer Konstanten:
class Mathe
{
    public const double Pi = 3.14159
}
                            

Verwenden der Deklarierten Konstanten:
double umfang = 2 * Mathe.Pi * 5;
Console.WriteLine(umfang);


Operatoren

Es gibt 9 verschiedene Arten von Operatoren:

  • Arithmetische Operatoren
  • Vergleichsoperatoren
  • Logische Operatoren
  • Zuweisungsoperatoren
  • Inkrement- und Dekrementoperatoren
  • Bedingungsoperator
  • Typumwandlungsoperatoren
  • Null-Operatoren
  • Bitweise Operatoren (Fortgeschritten)

Arithmetische Operatoren

Operator Bedeutung Beispiel Ergebnis
+ Addition 5 + 3 8
- Subtraktion 10 - 4 6
* Multiplikation 6 * 7 42
/ Division 10 / 2 5
% Modulo (Rest) 10 % 3 1 

int a = 10;
int b = 3;
int summe = a + b; // Ausgabe : "13" , denn 10 + 3 = 13
int rest = a % b; // Ausgabe : "1" , denn 10 / 3 = 3 Rest:1
double Kommazahl= a / (double)b; // Ausgabe : 3,333333335

Man kann also mit dieser Art von Operatoren im Programm rechnen.

Vergleichsoperatoren

Operator Bedeutung Beispiel Ergebnis
== Gleichheit 5 == 5 true
!= Ungleichheit 5 != 3 true
> Größer als 5 > 3 true
<< /td> Kleiner als 2 < 5 true
>= Größer oder Gleich 5 >= 5 true
<=< /td> Kleiner oder Gleich 3 <= 7 true

Mit Vergleichsoperatoren kann man also prüfen, ob etwas Gleich, Größer, Ungleich oder Gleich ist.

Logische Operatoren

Operator Bedeutung Beispiel Ergebnis
&& UND true && false false
` ` ODER
! NICHT !true false

Logische Operatoren verknüpfen boolesche Werte!

Zuweisungsoperatoren

Operator Bedeutung Beispiel Ergebnis
= Zuweisung x = 5 x ist 5
+= Addire und weise zu x += 3 x = x + 3
-= Subtrahire und weise zu x -= 2 x = x - 2
*= Multipliziere und weise zu x *= 4 x = x * 4
/= Dividiere und weise zu x /= 2 x = x / 2

Zuweisungsoperatoren weisen also Variablen, etc. einen Wert zu.
Diese werden aoch oft mit einer Operation(+, -, *, /) kombiniert.

Inkrement und Dekrementoperatoren

Operator Bedeutung Beispiel Ergebnis
++ Erhöhen x++ Nachher x = x + 1
-- Verringern x-- Nachher x = x - 1

Diese Art von Operatoren Erhöhen oder Verringern also den Wert der Variable um 1.

Kontrollstrukturen

Es gibt 4 Verschiedene Arten von Kontrollstrukturen:

  • Für Entscheidungen: if, else if, else
  • Zum Vergleichen von Mustern: switch
  • Schleifen: for, while, do-while, foreach
  • Zum Schleifen kontrollieren: break, continue

If, else if, else


int alter = 17;
if (alter >= 18)
{
    Console.WriteLine("Volljährig!")
}
else if (alter >= 16)
{
    Console.WriteLine("Fast Volljährig!")
}
else 
{
    Console.WriteLine("Noch zu Jung")
}

Je nachdem ob die Bedingung true oder false ergibt, wählt das Programm einen anderen Weg.
