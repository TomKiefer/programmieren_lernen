Über HTML Struktur Attribute Tags

Was ist HTML?

HTML steht für HyperText Markup Language und ist die Sprache, mit der Webseiten aufgebaut werden. Sie beschreibt, wie Inhalte im Browser dargestellt werden.

Wofür benutze ich HTML?

Html wird verwendet, um Webseiten zu erstellen und deren Inhalte zu strukturieren. Damit legt man fest, welche Elemente wie Texte, Bilder, Links oder Listen auf einer Seite erscheinen. Es bildet die Grundlage jeder Website und sorgt dafür, dass Browser den Inhalt richtig darstellen können.

Wie installiere ich notwendige Programme?

Man benutzt für HTML oft ein Programm wie Visuel Code Studios oder Notepad++. Diese Programme sind kostenlos und können einfach aus dem Internet heruntergeladen und installiert werden. Hier ist ein Link zum Dowload von Visual Code Studio Klick mich zum download.

Struktur einer HTML-Datei

<!DOCTYPE html> <html lang="de"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Seitentitel</title> </head> <body> <h1>Hallo Welt</h1> <p>Dies ist ein Absatz.</p> </body> </html>

Erklärung der einzelnen Zeilen:

  • <!DOCTYPE html> – sagt dem Browser, dass es sich um ein HTML-Dokument handelt.
  • <html lang="de"> – Beginn der HTML-Seite; das Attribut lang="de" zeigt, dass die Seite auf Deutsch ist.
  • <head> – enthält Informationen über die Seite (z. B. Titel, Zeichensatz, Verknüpfungen).
  • <meta charset="UTF-8"> – legt fest, dass Sonderzeichen wie Umlaute korrekt dargestellt werden.
  • <title> – bestimmt den Titel, der im Browser-Tab angezeigt wird.
  • <body> – hier steht der sichtbare Inhalt der Webseite.
  • <h1> – eine große Überschrift, in diesem Beispiel „Hallo Welt“.
  • </body> </html> – das / bedeutet, dass man etwas schließt.

HTML-Attribute

Was sind Attribute?

Attribute sind zusätzliche Informationen, die HTML-Tags mehr Details geben. Sie werden innerhalb des öffnenden Tags hinzugefügt und bestehen aus einem Namen und einem Wert.

Beispiele für Attribute:

  • href – wird im <a>-Tag verwendet, um die URL eines Links anzugeben.
    Beispiel: <a href="https://www.example.com">Besuche Seite</a>
  • src – wird im <img>-Tag verwendet, um die Quelle eines Bildes anzugeben.
    Beispiel: <img src="bild.jpg" alt="Ein Beispielbild">
  • class – wird im <img>-Tag verwendet, um eine alternative Textbeschreibung für ein Bild bereitzustellen.
    Beispiel: <p> class="highlight" > ist ein Satz mit einer CSS-Klasse </p>

    id – Gibt eine eindeutige ID für das Element an.
    Beispiel: <p id="einleitung">Das ist ein Satz mit einer eindeutigen ID.</p>
    lang – wird im <img>-Tag verwendet, um eine alternative Textbeschreibung für ein Bild bereitzustellen.
    Beispiel: <img src="logo.png" alt="HTML Bild">
    alt – wird im <img>-Tag verwendet, um eine alternative Textbeschreibung für ein Bild bereitzustellen.
    Beispiel: <img src="logo.png" alt="HTML Bild">
  • style – ermöglicht das Hinzufügen von CSS-Stilen direkt zu einem HTML-Element.
    Beispiel: <p style="color: blue;">Blauer Text</p>

HTML-Tags

Alle HTML-Tags:

  • <!DOCTYPE> – Definiert den Dokumenttyp (HTML-Version)
  • <html> – Wurzelelement des HTML-Dokuments
  • <head> – Enthält Metadaten über das Dokument
  • <title> – Titel der Webseite (im Browser-Tab)
  • <body> – Sichtbarer Hauptinhalt der Seite
  • <h1>–<h6> – Überschriften verschiedener Ebenen
  • <p> – Absatz
  • <a> – Hyperlink
  • <img> – Bild
  • <div> – Allgemeiner Container für Inhalte
  • <span> – Inline-Container für Text
  • <br> – Zeilenumbruch
  • <hr> – Horizontale Linie / Trenner
  • <header> – Kopfbereich eines Abschnitts
  • <footer> – Fußbereich eines Abschnitts
  • <nav> – Navigationsbereich
  • <main> – Hauptinhalt der Seite
  • <section> – Thematischer Abschnitt
  • <article> – Selbstständiger Artikel
  • <aside> – Seitenleiste oder ergänzende Inhalte
  • <form> – Formular
  • <input> – Eingabefeld
  • <button> – Schaltfläche
  • <label> – Beschriftung für ein Formularfeld
  • <select> – Dropdown-Menü
  • <option> – Eintrag in einer Auswahlliste
  • <textarea> – Mehrzeiliges Eingabefeld
  • <table> – Tabelle
  • <tr> – Tabellenzeile
  • <td> – Tabellenzelle
  • <th> – Tabellenkopfzelle
  • <thead> – Tabellenkopf
  • <tbody> – Tabelleninhalt
  • <tfoot> – Tabellenfuß
  • <ul> – Ungeordnete Liste
  • <ol> – Geordnete Liste
  • <li> – Listenelement
  • <figure> – Container für Bild + Beschreibung
  • <figcaption> – Bildunterschrift
  • <audio> – Audioelement
  • <video> – Videoelement
  • <source> – Medienquelle für Audio/Video
  • <track> – Untertitelspur für Videos
  • <canvas> – Zeichenfläche für Grafiken
  • <svg> – Vektorgrafik
  • <iframe> – Eingebettete Seite
  • <meta> – Metadaten im Head
  • <link> – Verknüpft CSS oder andere Ressourcen
  • <script> – Eingebettetes oder externes JavaScript
  • <style> – Eingebettetes CSS
  • <strong> – Starker, hervorgehobener Text
  • <em> – Betonter Text
  • <small> – Kleiner Text
  • <mark> – Hervorgehobener Text
  • <code> – Quelltext- oder Codeabschnitt
  • <pre> – Vorgeformter Text (Zeilenumbrüche bleiben)
  • <blockquote> – Zitatblock
  • <q> – Kurzes Inline-Zitat
  • <abbr> – Abkürzung mit Erklärung
  • <address> – Kontaktinformation
  • <time> – Zeit- oder Datumsangabe
  • <progress> – Fortschrittsanzeige
  • <meter> – Messwert innerhalb eines Bereichs
  • <details> – Ausklappbarer Informationsblock
  • <summary> – Überschrift für ein <details>-Element
  • <dialog> – Dialog- oder Pop-up-Fenster
  • <template> – Vorlage für wiederverwendbare Inhalte
  • <slot> – Platzhalter in Web Components
  • <noscript> – Inhalt, falls JavaScript deaktiviert ist
  • <data> – Maschinell lesbare Daten
  • <output> – Ausgabefeld in Formularen
  • <base> – Basis-URL für relative Links
  • <b> – Fett, ohne Betonung
  • <i> – Kursiv, ohne Betonung
  • <u> – Unterstrichener Text
  • <s> – Durchgestrichener Text
  • <sup> – Hochgestellter Text
  • <sub> – Tiefgestellter Text
  • <cite> – Zitatquelle
  • <dfn> – Definition eines Begriffs
  • <kbd> – Tastatureingabe
  • <var> – Variable in Code oder Formeln
  • <del> – Gelöschter Text
  • <ins> – Eingefügter Text
  • <map> – Bildkarte für klickbare Bereiche
  • <area> – Klickbarer Bereich in einer Bildkarte
  • <object> – Eingebettetes Objekt (z. B. PDF)

  • <embed> – Eingebettete Inhalte (z. B. Medien)
  • <picture> – Bild mit mehreren Quellen
  • <source> – Quelle für Medien (<audio>, <video>, <picture>)
  • <track> – Untertitelspur für Medien
  • <colgroup> – Spaltengruppe in Tabellen
  • <col> – Definiert Spaltenattribute in Tabellen
  • <fieldset> – Gruppiert Formularfelder
  • <legend> – Beschriftung für <fieldset>
  • <optgroup> – Gruppiert Optionen in einem <select>
  • <bdi> – Text mit anderer Schreibrichtung
  • <bdo> – Überschreibt Textausrichtung
  • <wbr> – Zeilenumbruch-Vorschlag
